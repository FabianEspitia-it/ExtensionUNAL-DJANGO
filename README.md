# UNAL EXTENSION (Is'nt Responsive)

This is a University project made with the Django Framework that tries to representante the National University Extension module. This is the Databases final project.

## Tech Stack

**Client:** HTML5, CSS3

**Server:** DJANGO

**DB:** MYSQL
